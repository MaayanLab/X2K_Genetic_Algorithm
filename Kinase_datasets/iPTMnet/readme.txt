iPTMnet PTM and Protein Information data files for release 4.1
===============================================================

This directory contaitains two tab-delimited files: 

1) ptm.txt -- PTM information
	ptm_type 
	source 
	substrate_AC 
	substrate_genename
	organsim
	site 
	enzyme_AC 
	enzyme_genename
	note 
	pmid

2) score.txt -- score information
        substrate_AC
        site
        enzyme_AC
        ptm_type
        score

3) protein.txt --  Protein Information
	UniProtAC
	UniProtID
	protein_name
	genename
	organsim
	PRO_id
	SwissProt/TrEmbl
